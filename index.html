Double Materiality Assessment

For CSRD preparation

Run a structured double materiality assessment

This project helps companies document CSRD and ESRS double materiality work in a practical browser-based tool. It supports impact and financial materiality scoring, stakeholder and evidence tracking, threshold decisions, and export of the matrix and report-ready topic pages.

Open assessment tool Read guides

What it does

The tool starts from the ESRS topic universe and lets users add, adapt and score sustainability matters for their own company, value chain and stakeholder context.

Exports

Users can export a PDF report, SVG matrix or PNG matrix. The PDF places the matrix first, followed by detailed topic pages suitable for inserting into a sustainability report.

Important note

The website is a documentation aid for CSRD/ESRS work. It does not replace legal review, management judgement, assurance procedures or the complete ESRS sustainability statement.
